<h3>Setup</h3>
<br>
1. clone https://github.com/erthru/Badminton-Racket.git <br>
2. cd Badminton-Racket <br>
3. composer install <br>
4. npm install <br>
5. php artisan migrate <br>
6. php artisan serve <br>
7. npm run watch <br>
<br><br>
API Documentation on : /docs (ex: http://localhost:8000/docs)
